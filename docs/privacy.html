Privacy Policy

Last updated: January 2026

Summary

SaneBar does not collect, transmit, or store any personal data. Your menu bar configuration and preferences remain entirely on your Mac.

Data Collection

SaneBar does not collect:

  • Personal information
  • Usage analytics
  • Crash reports
  • Telemetry of any kind

Local Storage

SaneBar stores the following data locally on your Mac:

  • Preferences: Standard macOS UserDefaults
  • Icon layout: Your menu bar icon arrangement and visibility settings

This data never leaves your computer.

Permissions

SaneBar requires:

  • Accessibility: To manage menu bar icon visibility and ordering
  • Touch ID: Optional, for locking sensitive icons behind biometric authentication

These permissions are used solely for the app's core functionality. No data from these permissions is ever transmitted.

Network Access

SaneBar accesses the network only when checking for app updates via Sparkle (can be disabled in Settings). No personal data is transmitted.

Third-Party Services

SaneBar uses:

  • Sparkle: For checking app updates. Only checks for updates from sanebar.com. No personal data is transmitted.

100% Transparent Code

Every line of SaneBar's code is public. Review it at github.com/sane-apps/SaneBar.

Contact

Questions about privacy? Email hi@saneapps.com or open an issue on GitHub.
